🔑 Personal Access Token Usage

To push code securely to GitHub:

  • Go to GitHub Settings → Developer Settings → Tokens
  • Generate a new token with repo access
  • Use it like this:
https://yourusername:yourtoken@github.com/yourrepo.git

🌿 Branch Merging Task

To create and merge branches:

  • git checkout -b feature-branch – create branch
  • git add . & git commit -m "your msg"
  • git push origin feature-branch
  • Open PR or use git merge in main branch

⚙️ Basic CI/CD Flow (Concept)

  • Push to GitHub triggers GitHub Actions
  • Run test/build/deploy steps
  • Deploy to staging or production
